## Overview of the steps

- install a better keyboard for coding and the command line
- install termux
  - install from FDroid
  - grant fs access
  - setup openssh
- install neovim
  - bootstrap neovim config with kickstart.nvim
  - install the correct packages
- install love2d
- make a first love2d thing
